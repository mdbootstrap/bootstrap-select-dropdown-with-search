
Responsive Select dropdown with search built with Bootstrap 5. Helps you find the options you are looking for by searching.

Check out [Bootstrap Select dropdown with search Documentation](https://mdbootstrap.com/docs/standard/extended/select-dropdown-with-search/) for detailed instructions & even more examples.
